# Option Visualizer Backend

## Running the Backend
To start the backend server, run:
```bash
uv run uvicorn main:app --reload --port 8000
```
## Stopping the Backend
To stop the backend server manually, press `Ctrl+C` in the terminal window where it is running.


## Running the Frontend
To start the frontend server, navigate to the `frontend` directory and run:
```bash
npm run dev
```
## Stopping the Frontend
To stop the frontend server manually, press `Ctrl+C` in the terminal window where `npm run dev` is running.
